# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview
This project is an end-to-end Sales Analytics solution designed to analyze business performance across **Revenue, Product, Customer, and Growth dimensions**.

The analysis was performed using **Python (Data Cleaning), SQL Server (Analysis), and Power BI (Visualization)** to transform raw multi-source data into actionable business insights.

---

## 🛠️ Tools & Technologies
- **Python (Pandas)** – Data Cleaning & Transformation  
- **SQL Server** –  Advanced Analysis  
- **Power BI** – Data Modeling ,Dashboard & Visualization  
- **Excel (Source Data)**  

---

## 📂 Dataset Description
- 64K+ records across multiple Excel sheets  
- Sales transactions, customers, products, geography  
- Data cleaned, merged, and transformed into a single structured dataset  

---

## 🔄 Data Pipeline
1. Cleaned and merged 6 Excel datasets using Python  
2. Fixed structural issues (misaligned headers, missing values)  
3. Created derived features:  
   - Profit  
   - Total Cost  
   - Profit Margin %  
4. Exported final dataset to SQL Server  
5. Performed advanced SQL analysis  
6. Built interactive Power BI dashboard  

---

## 📊 Dashboard Structure

### 1️⃣ Executive Overview
- Total Revenue: ~$1.2B  
- Profit Margin: ~37%  
- Channel & Region performance  
- Monthly trend analysis  

---

### 2️⃣ Product Performance
- Top products drive majority of revenue  
- Product portfolio follows **Pareto distribution (~70/20/10)**  
- Export channel shows higher margins  
- Identified high-margin vs high-volume products  

---

### 3️⃣ Customer Segmentation (RFM Analysis)
- Segmented customers into:
  - Champion  
  - Loyal Customer  
  - Potential Loyalist  
  - Developing  
- Found strong retention and repeat purchase behavior  
- Revenue driven primarily by high-frequency customers  

---

### 4️⃣ Sales Growth Analysis
- Revenue stable (~$23M–$26M monthly)  
- Clear seasonal pattern:
  - Dip: Feb–Apr  
  - Peak: May & August  
  - Stable: Q4  
- Rolling average confirms mature business trend  

---

## 🔍 Key Business Insights

- Revenue is highly concentrated in top products and customers  
- Wholesale drives volume (~54%), Export drives profitability  
- West region dominates, Northeast underperforms  
- Customer base is healthy with low churn signals  
- Growth opportunity lies in **upselling and increasing frequency**, not demand expansion  

---

## 📈 Advanced Analysis Performed

- RFM Segmentation  
- ABC Classification (Pareto Analysis)  
- Customer Lifetime Value (CLV)  
- YoY & MoM Growth Analysis  
- Rolling Average Trend Analysis  
- Channel × Product Profitability Matrix  

---

## 💡 Business Recommendations

- Expand Export channel to improve margins  
- Reduce dependency on top products  
- Target mid-tier customers for upselling  
- Improve low-season performance (Feb–Apr)  
- Optimize low-performing products
